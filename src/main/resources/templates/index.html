How to Create a Dashboard Using Tableau®


Interested in learning how to create dashing dashboards in Tableau to showcase your amazing analyses? Here are some suggestions to help you get started.

You can download Tableau Desktop here. Using Tableau Desktop gives you privacy and you can post your completed dashboard to Tableau Public if you so desire.

Alternatively, you can create a dashboard without downloading anything using Tableau Online. Never use this option (or post to Tableau Public) if you are using proprietary (company) data.
